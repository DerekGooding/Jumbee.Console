# <a id="Jumbee_Console_RunChart"></a> Class RunChart

Namespace: [Jumbee.Console](Jumbee.Console.md)  
Assembly: Jumbee.Console.dll  

A live multi-series time chart with a legend — a streaming line <xref href="Jumbee.Console.Plot" data-throw-if-not-resolved="false"></xref> on the left and a per-series
stat readout (name + current / delta / max / min) on the right, in the style of a monitoring "run chart".

## Remarks

Add a series with <xref href="Jumbee.Console.RunChart.AddSeries(System.String%2cJumbee.Console.Color)" data-throw-if-not-resolved="false"></xref> and feed it through the returned <xref href="Jumbee.Console.RunSeries" data-throw-if-not-resolved="false"></xref> handle
(<xref href="Jumbee.Console.RunSeries.Push(System.Double)" data-throw-if-not-resolved="false"></xref>); the data flows through a stationary strip axis (fixed X window) so the frame stays
put and the values scroll through it. Pure composition over <xref href="Jumbee.Console.Plot" data-throw-if-not-resolved="false"></xref> — it adds a legend, not a new
rendering path.
