## Remarks

Closed, it shows the current value with a ▼ marker; clicking it (or Enter/Space while
focused) opens its options in the ambient <xref href="Jumbee.Console.UI.Overlay" data-throw-if-not-resolved="false"></xref>. By default the list opens below the control,
flipping above when there isn't room (see <xref href="Jumbee.Console.Select.PopupPosition" data-throw-if-not-resolved="false"></xref>). Choosing an option (click or Enter)
commits it; Escape or a click outside cancels.
