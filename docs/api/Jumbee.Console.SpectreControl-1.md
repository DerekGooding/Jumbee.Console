## Remarks

Uses an <xref href="Jumbee.Console.AnsiConsoleBuffer" data-throw-if-not-resolved="false"></xref> to render the control to a buffer.
Public property setters and methods that change a control's visual state should call <xref href="Jumbee.Console.Control.Invalidate" data-throw-if-not-resolved="false"></xref> to request a re-render on the next UI update tick.
Non-atomic changes to the wrapped content (e.g. mutating its collections) should go through <xref href="Jumbee.Console.SpectreControl%601.UpdateContent(System.Action%7b%600%7d)" data-throw-if-not-resolved="false"></xref>,
which applies the change on the UI thread so it never races with rendering.
