

# Jumbee.Console,

## About
[Jumbee.Console](https://github.com/allisterb/Jumbee.Console) is a .NET library for advanced TUIs that focuses on performance and ease-of-use. Inspired by libs like [ratatui](https://ratatui.rs/) and [Textual](https://textual.textualize.io/), it tries to provide a high-performance modern TUI library that is easy-to-use with idiomatic .NET GUI and Task patterns, while flexible enough to create different types of TUI applications from news readers to animated dashboards to IDEs to agent harnesses to graphics apps.


## Features

* 100% managed AOT-compliant code.
* Retained-mode GUI framework with a modern API designed to be easy to use and extend.
* Sub-ms frame rendering times and minimal CPU consumption even with complex displays like multi-tab document editing and syntax highlighting.
* Full mouse support via SGR 1006
* Supports both fixed-width layouts like `Grid` and flexible, resizable layouts like `DockPanel`, `HorizontalStack`, `VerticalStack`, resizable `SplitPanel`.
* Large set of controls with common GUI elemnts: menus, buttons, trees, text inputs with autocomplete, modal dialog windows, etc....    
* Control frames support adornments like titles, borders, margins, scrollbars
* Cross-platform 100% managed code terminal-emulator
* Muti-tab editor that supports C#, JavaScript, C++, Markdown + a dozen other languages
* Split-pane interactive editors with preview for Markdown, AsciiDoc, Mermaid documents, Mermaid embedded in Markdown
* Many different types of plots and graphs and visualizations and graphics with support for animation
* Flexible themes that support styling both colors and glyphs independently
