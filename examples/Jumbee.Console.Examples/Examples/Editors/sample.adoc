= AsciiDoc Viewer
:author: Jumbee.Console
:toc:

A read-only, scrollable *AsciiDoc* viewer built on the pure-managed
https://github.com/sonatique/Adoc.Net[AdocNet] parser, rendering each AST node
as a Spectre.Console renderable.

== Inline Formatting

This paragraph shows *bold*, _italic_, and `monospace` text, a
https://asciidoctor.org[link], and a cross-reference all styled inline.

== Admonitions

NOTE: Admonitions render as bordered panels with an accent colour and glyph.

TIP: Wrap the viewer in a ControlFrame for a border, title and scrollbar.

WARNING: Very tall documents clip at the bottom row cap.

== Source Blocks

[source,csharp]
----
var viewer = new AsciiDocViewer(File.ReadAllText("doc.adoc"))
    .WithFrame("AsciiDoc");
ui.Add(viewer);
----
